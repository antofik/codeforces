Пусть

  • v(n) — наибольшее простое число, не превосходящее n;
  • u(n) — наименьшее простое, строго большее, чем n.

Найдите .

Входные данные

В первой строке задано целое число t (1 ≤ t ≤ 500) — количество тестов. В каждой из следующих t строк входных данных вводится целое число n (2 ≤ n ≤ 109).

Выходные данные

Выведите t строк: в i-й из них должен содержатся ответ на i-й тест в виде несократимой дроби «p/q», где p, q — целые числа, q > 0.
